สวัสดีค่ะชื่ออิงอิง มาจาก สว.๒

การครั้งนึงนานมาแล้วมีกระต่ายตัวนึงเจอเต่าที่ริมสระ เต่านั้นเดินลงไปแช่น้ำ กระต่ายเห็นเลยหัวเราะด้วยความดูถูก เต่าโมโหแล้วท่ากระต่ายวิ่งแข่งกัน ถ้าใครชนะจะต้องเลี้ยงชานมไข่มุก พอกระต่ายได้ยินดังนั้น กระต่ายตอบตกลงทันที พอมาถึงวันเเข่ง สัตว์นานาชนิด มาดูการแข่งขันกันมากมาย เมื่อมาถึงเส้น start เต่าบอกกระต่ายว่า ในป่าแห่งนี้มี ศาลเจ้าอยู่ที่นึง ถ้าขออะไรจะได้เป็นจริงหมด กระต่ายเชื่อจึงรี
